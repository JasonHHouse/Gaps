Configuring Gaps for Plex

We'll need a few pieces of information to work with Plex. Once you have all those, enter them below and we'll keep moving forward.

IP Address
Use your network IP address. localhost and 127.0.0.1 will not work for Plex and Gaps in Docker. Use ifconfig/ipconfig to find that in a terminal or command window.
Port
Defaults to 32400
Your Personal Plex Token
If you do not know already it's easy to find. Plex has a great write up here about how to find your token.
SSL/TLS
Is your Plex configured to use https? If you don't know, leave this unchecked.
